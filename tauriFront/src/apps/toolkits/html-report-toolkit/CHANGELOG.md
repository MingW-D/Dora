# 更新日志

## v2.0.0 - Studio集成版本

### ✨ 新功能
- 🖥️ **Studio预览**：HTML报告直接在Studio面板中显示
- 💾 **下载按钮**：用户可手动选择下载报告
- 🔍 **实时预览**：无需保存文件即可查看报告效果
- 📁 **自定义保存**：下载时可选择保存位置和文件名

### 🔄 功能变更
- ❌ **移除自动保存**：不再自动保存到固定目录
- ✅ **用户控制下载**：改为用户主动触发下载
- 🎨 **Studio界面**：新增专用的htmlReport显示模式

### 🛠️ 技术改进
- 使用`iframe`的`srcdoc`属性直接渲染HTML
- 通过Blob API实现客户端下载
- 优化Studio面板交互体验
- 简化文件权限管理

### 🐛 修复
- 解决Tauri文件系统权限限制问题
- 优化错误处理和用户体验
- 移除对受限目录的依赖
